# Computer Graphics  
### Vector Dot Product  
  a.b = b.a  
  a.b = ||a|| ||b|| cos (0)  
  
### Vector Cross Product  
  a x b = -b x a  
  ||a x b|| = ||a|| ||b|| sin (0)  
  
#### Properties  
  X x Y = +Z  
  Y x X = -Z  
  Y x Z = +X  
  Z x Y = -X  
  Z x X = +Y  
  X x Z = -Y  
  
#### Adding and Subtracting Vectors and Points
*  A point plus a vector will result in a point
*  A vector plus a vector will result in a vector
